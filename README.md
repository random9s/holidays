# holidays

Usage: ./main [year]

# Description
Generates holiday list for the given year for USA/Canada Bank Holidays

Holidays included are:

USA:
	New Years Day
	MartinLuther King Day
	Presidents Day
	Memorial Day
	Juneteenth
	Independence Day
	Labor Day
	Indigenous Peoples Day
	Veterans Day
	Thanksgiving Day
	Christmas Day
	Gregorian Easter

Canada:
	New Years Day
	Good Friday
	Gregorian Easter
	Victoria Day
	Canada Day
	Labour Day
	Truth And Reconciliation Day
	Thanksgiving Day
	Remembrance Day
	Christmas Day
	Boxing Day
